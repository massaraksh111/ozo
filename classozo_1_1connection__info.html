template<typename OidMap = empty_oid_map, typename Statistics = no_statistics>
class ozo::connection_info< OidMap, Statistics >

Connection information.

This type is a basic ConnectionSource implementation. This source allows to establish connection via connection string specified.

Template Parameters
OidMap— oids map type which defines user types are used within this connection.
Statistics— statistics type which defines statistics is collected for this connection.
